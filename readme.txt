Au niveau de notre Application, 

Ce qui marche :
-Le basculement d'une fenêtre à une autre
-La fermeture d'une page
-Le retour en arrière d'une page
-La partie Liste Mémoire
-La partie SQL (en théorie)
-La création des éléments
-La suppression des éléments dans une table
-L'affichage des éléments dans une table 
-L'affichage dans les choicebox
-Lorsque que l'on créee un élément, les champs se vident
-Si il manque un attribut dans la création de l'élément, l'application nous le rappelle en affichant un message d'erreur avec l'attribut manquant
-La clarté du code, il est expliqué par des commentaires
-Les boutons supprimer, modifier, ajouter sont désactivés
-Le bouton ajouter s'active lorsque l'on rempli les champs
-Le bouton modifier s'active lorsque l'on séléctionne un élément dans le tableau
-Le bouton supprimer s'active lorsque l'on séléctionne un élément dans le tableau
-Le bouton rechercher s'active lorsque l'on écrit dans la zone de texte
-Les éléments dans le tableau se trient par ordre croissant ou décroissant (alphabétique et numérique)
-On peut basculer d'un Menu à l'autre facilement

Ce qui ne fonctionne pas : 

-La recherche d'un élément 
-La connexion SQL (Erreur connexionThe server time zone value 'CEST' is unrecognized or represents more than one time zone. You must configure either the server or JDBC driver (via the 'serverTimezone' configuration property) to use a more specific time zone value if you want to utilize time zone support.)
-La modification d'un élément

Nous avons fait une esthétique dans le thème de Noël afin que l'application soit plus agréable à regarder et utiliser.
